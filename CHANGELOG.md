# CHANGELOG

## v1.2 · 2026-04-18

### 核心改动
- 保留 8-Type 数量，不扩容
- 保留三问诊断法
- 保留宏观 Regime 与三条传导链
- 保留远端锚 vs 近端溢价
- 保留 conviction = 结构 + 情绪 + 转换距离

### 结构系统改动
- 将 Type 与方向拆开：Type 只回答结构，方向统一用 `+ / - / 0`
- 将 Type 5 从主结构改为风险标签（overlay）
- 明确 Type 7 优先级高于 Type 4
- 新增“路径标签”，用于描述价格走法
- 新增判定优先级树，减少混合态歧义

### 命名标准化
- Type 1：现货主导型
- Type 3：共振趋势型
- Type 6：缓慢出清型
- Type 7：级联踩踏型

### 输出层升级
旧版：
- 当前 Type

新版：
- 路径标签
- 主结构
- 风险标签

### 为什么这样改
- 解决方向不对称问题
- 解决 Type 5 与其它 Type 抢主结构的问题
- 解决 Type 4 / Type 7 重叠问题
- 让模板更接近程序化状态机输出
